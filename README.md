# Aurora Grand Hotel Website

A vanilla HTML/CSS/JS hotel landing page with a simple Node.js backend.

## Run locally

1. Open a terminal in `c:\Users\alema\Desktop\Bens`
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the site:
   ```bash
   npm start
   ```
4. Open `http://localhost:3000`

## Features

- Hero section with booking CTA
- About, services, testimonials, and contact sections
- Responsive navigation
- Contact form that posts to `/api/reservation`
- Node.js backend saves requests to `bookings.json`
